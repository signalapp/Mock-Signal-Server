# Signal Mock Server

## Overview

This npm module is a mock implementation of Signal Server, currently only used
in [Signal Desktop integration tests][0]. Public API surface area can be found at
[`src/api`][1].

## Installation

```sh
npm install --dev @signalapp/mock-server
```
